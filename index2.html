ROTEADOR WIFI - HOTSPOT

ROTEAMENTO DESLIGADO

Ao ativar o hotspot com compartilhamento de VPN do seu celular, um IP local é gerado no seu dispositivo. Este IP é usado como proxy no dispositivo conectado à sua rede Wi-Fi.

SIGA ESTES PASSOS IMPORTANTES

1 - Ligue o roteador do celular primeiro.

2 - Ative o compartilhamento de VPN.

3 - Use o botão para controlar o roteador.

4 - Encontre o segundo IP na barra de notificações.

5 - Copie o segundo IP e use como proxy no dispositivo conectado à rede.

Lembre-se: O segundo IP é seu proxy. Isso garantirá o correto funcionamento do compartilhamento de VPN via hotspot do celular.
